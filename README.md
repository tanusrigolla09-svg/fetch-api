# Fetch & Display Data from a Public API

Fetches user data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users) and displays it in a styled card grid.

## Features
- Fetch API with async/await and error handling
- Live search and sort (by ID, Name, Company)
- User detail modal with full info + Google Maps link
- Dark / Light theme toggle
- Skeleton loading state + reload button

## Tech Used
HTML · CSS · Vanilla JavaScript · Fetch API
